先给你**核心定位**，避免学错方向：

> **AI 应用开发工程师 ≠ 算法训练工程师**
> 
> 不用啃高数、不用训练大模型、不用硬核深度学习炼丹。
> 
> 核心工作：**基于现成开源 / 闭源大模型，做 RAG 知识库、AI 智能体 Agent、私有化部署、业务系统 AI 集成、插件开发、企业级 AI 落地**，是零基础转行 AI**门槛最低、岗位最多、落地性最强**的方向。

## 整体学习周期

- 全职脱产：**3~5 个月** 可独立上岗
- 业余下班学习：**6~8 个月**
- 核心逻辑：**先懂概念→会编程→掌握 AI 核心技术→工具框架实操→做完整项目→工程化部署→求职**

---

# 第一阶段：行业 & 基础概念扫盲（1 周・理解为主）

## 1. 必懂基础概念（零基础必记）

1. 基础名词
    
    AI、生成式 AI、大语言模型**LLM**、多模态模型、Token、上下文窗口、模型幻觉
2. 岗位区分

- 算法工程师：训模型、调参、高强度数学，不适合零基础
- **AI 应用开发**：调用模型、二次开发、落地业务、工程化，你的目标

3. 核心技术名词（后续天天用）

- Prompt 提示词：控制大模型输出规则
- **Embedding 向量嵌入**：文字转数字向量，实现语义检索
- **RAG 检索增强生成**：解决大模型幻觉、私有数据问答（核心核心）
- **Agent 智能体**：AI 自主思考、拆解任务、调用工具
- 轻量化微调 LoRA：小成本定制行业专属模型

4. 模型生态认知

- 闭源商用模型：通义千问、文心一言、讯飞星火、DeepSeek（靠 API 调用）
- 开源免费模型：阿里 Qwen、清华 GLM、Llama3、Mistral（可本地离线部署）

## 2. 行业场景了解

企业刚需场景：企业知识库问答、AI 办公助手、客服机器人、文档解析、自动化流程 Agent、本地私有化 AI、多模态图文理解。

---

# 第二阶段：刚需编程基础（3~4 周・实操优先）

AI 应用开发**唯一必学语言：Python**，不用学精通，只学 AI 开发刚需内容。

## 1. Python 核心必学（2 周）

### 必须掌握知识点

变量 / 数据类型、字典 & 列表（高频）、循环 / 条件判断、函数、基础类与对象、异常捕获、**JSON 数据解析**、文件读写。

### 工具安装

- 运行环境：Python 3.10 / 3.11（兼容性最强）
- 代码编辑器：**VS Code**（必学，轻量化、免费）

### 避坑

不用学爬虫、数据分析、可视化、机器学习算法，**只学 AI 开发刚需语法**。

## 2. 网络 & 接口基础（1 周）

1. 核心概念：HTTP/HTTPS、GET/POST 请求、API 接口、密钥 Key、JSON 数据格式
2. 核心能力：会通过代码调用第三方 AI 接口（所有大模型都是接口调用）

## 3. 极简后端基础（1 周）

只学轻量化框架，用来封装 AI 接口：

- **FastAPI**：快速搭建后端服务，企业 AI 项目通用
- 可选：Flask（更简单，入门过渡）

---

# 第三阶段：AI 核心底层原理（2 周・理解不深究）

**完全不用学高数、线代、概率论**，只理解逻辑，不做公式推导。

1. Transformer 架构：只懂「注意力机制」作用 —— 让模型理解上下文语义
2. 生成式逻辑：大模型逐 Token 生成文字的原理
3. 关键参数理解
    
    `temperature温度`（创意度）、`TopP`、上下文限制、模型上下文窗口
4. 向量基础：文本→向量→语义相似度匹配（RAG 核心原理）

---

# 第四阶段：核心技术专项突破（4 周・就业核心）

按**优先级从易到难**学习，全部动手敲代码。

## 1. Prompt 提示词工程（1 周・入门最快）

### 掌握内容

- 标准提示词结构：**角色 + 背景 + 任务 + 约束 + 输出格式**
- 进阶技巧：少样本 Few-Shot、思维链 CoT、任务拆分、内容风控

### 实操练习

文案生成、数据提取、表格整理、代码生成、行业问答提示词编写。

## 2. RAG 检索增强生成（2 周・企业刚需）

> 90% 企业 AI 应用都在用 RAG，转行**必吃透**

### 完整标准流程

文档上传 → 文本切片分割 → Embedding 向量化 → 向量入库 → 用户问题向量化 → 相似内容检索 → 大模型结合参考资料生成答案

### 必学知识点

文本分块策略、向量相似度、重排序 Rerank、RAG 常见问题优化（答案不准、上下文丢失）

## 3. AI Agent 智能体开发（1 周・高薪加分）

### 核心概念

工具调用、插件扩展、自主任务规划、短期 / 长期记忆、函数调用

### 实操能力

让 AI 自动调用计算器、读取本地文件、调用第三方接口、完成自动化任务。

## 4. 轻量化微调（选学）

只了解**LoRA 轻量化微调**即可，不用全量微调；

作用：定制行业专属话术、专属知识库模型，进阶加分。

---

# 第五阶段：主流工具 / 框架 / 模型 / 数据库（4 周・干活必备）

## 1. 两大核心开发框架（必学）

1. **LangChain**：生态最全，做 Agent、链式任务、多工具集成
2. **LlamaIndex**：专注知识库检索，RAG 场景优化更强

## 2. 向量数据库（RAG 必备）

表格

|类型|工具|适用场景|
|---|---|---|
|新手入门|Chroma、FAISS|本地免费、零配置、个人项目|
|企业生产|Milvus、Qdrant|高并发、海量文档、公司项目|

## 3. 本地开源模型部署工具（零基础神器）

1. **Ollama【首选】**：一键安装 Qwen/Llama/GLM 等开源大模型，CPU 就能跑，离线本地调用，零配置
2. ModelScope 魔搭、Hugging Face：国内 / 全球开源模型仓库，下载各类模型

## 4. 嵌入模型（RAG 专用）

必用：bge-m3、Qwen-Embedding、text-embedding（负责文本转向量）

## 5. 低代码 AI 前端框架

快速做网页版 AI 应用，不用学复杂前端：

- **Streamlit**、**Gradio**：几十行代码搭建 AI 网页界面

## 6. 统一调用 SDK

- OpenAI SDK：兼容所有国产开源大模型接口
- 大厂官方 SDK：阿里 DashScope、百度文心、讯飞星火

# 第六阶段：工程化 & 私有化部署（3 周・区分小白 / 工程师）

只会写 demo 无法就业，企业需要**能上线、能部署、私有化**的能力。

1. 本地离线部署：用 Ollama 部署开源大模型，实现无外网使用
2. 服务化封装：用 FastAPI 把 RAG/Agent 项目封装成 API 接口
3. Docker 基础：极简容器化，打包 AI 项目，一键部署
4. 权限 & 风控：内容脱敏、访问密钥、问答过滤（企业必备）

# 第七阶段：实战项目（贯穿全程・转行核心）

**零基础转行：项目＞证书＞理论**，所有项目代码上传 GitHub，作为求职作品集。

## 入门级项目（必做）

1. 基于 API 的 AI 对话助手（调用通义千问 / DeepSeek）
2. 本地私有知识库问答（PDF/Word/TXT 文档 RAG 问答）
3. 办公 AI 工具：文案生成、文档总结、数据提取

## 进阶级项目（求职主打）

1. 企业级 RAG 知识库系统（Milvus+LangChain + 文档解析）
2. 多模态 AI 助手（图文问答、图片 OCR、截图解读）
3. 轻量化 AI 客服机器人

## 高阶加分项目（高薪）

1. 插件化 AI 智能体（自动处理表格、邮件、业务流程）
2. 私有化离线 AI 平台（Docker 部署、多模型管理）

2. 面试核心知识点：RAG 原理、LLM 基础、向量库选型、Agent 机制、模型部署
3. 岗位投递关键词：
    
    `AI应用开发工程师、大模型应用开发、RAG开发工程师、AI智能体开发、私有化AI开发`

# 关键硬件要求（零基础友好）

1. 最低配置：8G 内存 + 固态硬盘（跑轻量 7B 模型）
2. 推荐配置：16G 内存（流畅本地部署开源大模型）
3. **无需高端显卡**：入门阶段纯 CPU 即可运行，降低转行成本

# 避坑指南（小白必看）

1. ❌ 不要先学深度学习、不要啃数学、不要尝试训练大模型
2. ❌ 不要只玩网页版 AI，必须动手写代码调用模型
3. ✅ 优先学**国产开源模型**（Qwen/GLM），贴合国内企业需求
4. ✅ 理论看懂就好，**70% 时间敲代码做项目**
