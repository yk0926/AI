## 机器学习 需要学到什么程度（必学范围）

### ✅ 必须学会（刚需，工作天天用）

1. 基础概念
    
    - 训练集 / 验证集 / 测试集
    - 过拟合、欠拟合
    - 归一化、标准化
    - 特征、标签、嵌入向量 Embedding
    
2. 常用经典算法**不用手推，但要懂作用**
    
    - 聚类、相似度计算（余弦相似度、欧式距离）—— **RAG 检索天天用**
    - 分类、回归任务区别
    
3. 评估指标懂概念即可
    
    - 准确率、召回率、F1、困惑度 Perplexity
        
        知道啥意思、什么时候用，不用自己实现计算。
    

### ✅ 了解即可（面试问到能说两句）

- 逻辑回归、决策树、随机森林、SVM 大概干什么的
- 传统机器学习特征工程大概思路

### ❌ 完全不用学

- 不用推导公式
- 不用手写梯度下降
- 不用深入各种机器学习变种算法细节

---

## 三、深度学习 需要学到什么程度（必学范围）

### ✅ 必须学会（刚需，不懂做不了 RAG/Agent）

1. 神经网络基础常识
    
    - 神经元、激活函数大概作用
    - 深层网络为什么比浅层强
    
2. **Transformer 核心必懂（重中之重）**
    
    不用看源码、不用推导；但必须懂：
    
    - 自注意力机制 **是干嘛的**：看上下文、关联前后语义
    - Token 是什么、模型按 Token 逐字生成
    - 上下文窗口 Context Window 限制是什么
    - 温度 temperature、top_p 对生成的影响原理
    
3. Embedding 嵌入原理
    
    - 文本怎么变成向量
    - 向量数据库为什么能做语义检索
    - RAG 整个链路的深度学习逻辑
    

### ✅ 了解即可

- CNN 图像识别大概用途
- RNN/LSTM 老模型是什么、为什么现在被 Transformer 淘汰
- 多模态模型（Qwen-VL、CLIP）图文对齐大概原理

### ❌ 完全不用学

- 不用反向传播数学推导
- 不用自己搭建 Transformer 网络
- 不用研究优化器、学习率调度、BatchNorm 细节
- 不用啃 PyTorch 底层源码

---

## 四、大模型相关 NLP 要学到什么程度

### ✅ 必懂

- 分词、Token 消耗原理
- 语义理解、语义相似度
- 对话上下文拼接原理
- 提示词为什么能影响输出

### ❌ 不用学

- 传统分词算法源码
- 句法分析、词性标注老式 NLP 全套

## 五、结合你是**Java 后端转行**的最佳学习标准

给你一个可以直接执行的标准：

1. **不碰高数、不碰公式推导、不啃论文**
    
2. 只学**概念、流程、作用、适用场景**
    
3. 能达到以下水平就够上岗：
    
    - 看得懂简单 PyTorch 微调示例代码
    - 能解释 RAG 底层向量语义检索原理
    - 能说清 Transformer 注意力机制是干嘛的
    - 知道过拟合、Token、Embedding、上下文窗口是什么
    - 能听懂算法同事术语、能对齐需求、能排查 AI 应用异常
    
4. 把节省下来的时间，重点学：
    
    Python + FastAPI + LangChain + LangGraph + RAG + Agent + 向量库 + Ollama/vLLM 部署
